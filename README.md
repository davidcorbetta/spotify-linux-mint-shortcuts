# Shortcuts to spotify media using linux mind.

## Shortcuts available to add on the keyboard in *"custom shortcuts"*:

**Play:**  
`run.sh "Play"`  
**Stop:**  
`run.sh "Pause"`  
**Next:**  
`run.sh "Next"`  
**Previous:**  
`run.sh "Previous"`  
**Volume +5%:**  
`run.sh "+"`  
**Volume -5%:**  
`run.sh "+"`

**Configure notification in run.sh:**  
`notification=true||false`
